## 🎯 使用場景

### 場景 1：日常卜卦
```
1. 選擇 Light 主題（白天）
2. 點擊「開始卜卦」
3. 輸入問題
4. 觀看動畫
5. 查看結果
6. 複製保存
```

### 場景 2：夜間使用
```
1. 選擇 Dark 主題（護眼）
2. 開始卜卦
3. 享受暗黑模式的舒適體驗
```

### 場景 3：分享結果
```
1. 完成卜卦
2. 點擊「複製結果」
3. 貼到聊天軟體
4. 與朋友討論
```

### 場景 4：建立日記
```
1. 每次卜卦後複製結果
2. 貼到筆記軟體
3. 添加個人感想
4. 定期回顧
```
